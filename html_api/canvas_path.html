canvas arc

기본용법

arc(x, y, radius, startAngle, endAngle, direction)

  • x와 y는 중심점의 좌표를 나타낸다.
  • radius는 반지름의 길이를 나타낸다.
  • startAngle은 처음 그리기 시작하는 각도이며 수평으로 오른쪽이 0도를 나타낸다.
    각도는 도(degree)를 사용하지 않고 라디언(radian)깂을 사용하으므로 Math.PI를 이용하여 계산한다.
    예를 들어 Math.PI는 180도를 나타내므로 45도를 표현하기 위해서는 Math.PI / 180 * 45와 같이 구할 수 있다.
  • direction은 그리는 방향이며 불리언값을 갖는다.
    true는 반시계 방향으로 그리고 false는 시계방향으로 그린다.
  • endAngle은 그리기를 종료하는 각도

주의사항

  • 새로운 path가 시작될 땐 다시 beginPath()를 써줘야한다.
